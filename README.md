# طلب سريع

تطبيق بسيط لعرض منتجين وإرسال الطلب عبر واتساب.

## تعديل رقم الواتساب
من داخل ملف `app.js` عدّل هذا السطر:

```js
const WHATSAPP_NUMBER = '972525699248';
```

## تعديل المنتجات
من داخل `app.js` عدّل مصفوفة `products`.

## إضافة الصور الحقيقية لاحقًا
حالياً التطبيق يستخدم Placeholder بسيط.
إذا أردت لاحقًا إضافة صور حقيقية، يمكن ربط كل منتج بصورة بسهولة داخل البطاقات.
